# TODO List

## Installation

Start with cloning this repo on your local machine:

```sh
$ git clone https://github.com/adamlin22/TODO-List.git
$ cd TODO-List
```

To install and set up the library, run:

React.js for the frontend:

```sh
$ npm install
```

Express.js for the backend:

```sh
$ cd backend
$ npm install
```

## Usage

### Serving the app

Frontend:

```sh
$ npm start
```

Backend:

```sh
$ cd backend
$ node app.js
```
